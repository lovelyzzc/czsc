# 主升浪追涨 CZSC 特征每日选股

## 工作流

执行以下三步，每步完成后再进入下一步：

### Step 1: 同步最新数据

```bash
PYTHONUNBUFFERED=1 /home/lovelyzzc/czsc/.venv/bin/python /home/lovelyzzc/czsc/scripts/_sync_daily_data.py
```

确认输出包含"完成"或"数据已是最新"后继续。

### Step 2: 运行每日扫描

```bash
PYTHONUNBUFFERED=1 /home/lovelyzzc/czsc/.venv/bin/python /home/lovelyzzc/czsc/.cursor/skills/surge-wave-stock-picker/scripts/daily_scan.py
```

扫描全 A 股约 10-15 秒，输出等权得分 >= 5 且处于"回调入场窗口"的标的。

### Step 3: 汇报结果

脚本会自动进行四维优先级评分并精选 Top 20 可入场标的。将精选结果呈现给用户：

| 列 | 说明 |
|---|---|
| 等级 | A(>=80可直接入场) / B(65-79等确认) / C(<65观察池) |
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 行业 | 所属行业 |
| 收盘价 | 最新收盘价 |
| 得分 | 等权总分（满分 7） |
| 优先级 | 综合评分（满分100），越高越优先入场 |
| MA10偏离% | 当前价 vs MA10 偏离度，负值=在MA10下方（最佳入场位） |
| 缩量比 | 近3日均量/20日均量，<0.8说明回调缩量（洗盘特征） |
| **推荐止损** | **SL2 笔结构止损价（最近向下笔低点）** |
| 止损幅度% | (收盘价 - 推荐止损) / 收盘价 |

等级说明：
- **A 级**（优先级 >= 80）：高确定性，可直接入场
- **B 级**（优先级 65-79）：关注观察，等尾盘确认或次日低开入场
- **C 级**（优先级 < 65）：观察池，等更好的回调位置

精选过滤条件：已回调到MA10 + 止损幅度3-30% + 当前得分仍>=5。

**止损穿透处理**：若当前价已跌破推荐止损位（止损幅度% <= 0），说明笔结构止损
已触发，该标的优先级直接归零、等级强制降为 C，不会出现在精选列表中。

## 特征打分说明（等权，满分 7）

| 编号 | 条件 | 含义 |
|---|---|---|
| S1 | 笔力加速 | 最近 3 向上笔力度递增 |
| S2 | 力度比 | 向上笔均力度 / 向下笔均力度 > 1.5 |
| S3 | 脱离中枢 | 收盘价 > 最近中枢 ZG x 1.2 |
| S4 | 低点抬升 | 最近 2 向下笔低点逐笔抬升 |
| S5 | MA扩散 | MA5>MA10>MA20 且散度 > 15% |
| S6 | DIF加速 | DIF > 0 且 DIF > 5日前 DIF |
| S7 | 涨幅确认 | 最近 20 日涨幅 > 15% |

入选阈值：等权总分 >= 5。

## B+D 组合策略（入场 + 退出）

### 入场规则（D: 回调入场）

信号触发（得分 >= 5）后不立即追高，而是等待 10 天内价格回调至 MA10 附近
（Close <= MA10 x 1.02）再入场。超过 10 天未回调则放弃。

### 退出规则（B: 混合退出）

三级退出机制，兼顾止损保护和利润锁定：

| 阶段 | 条件 | 退出规则 |
|---|---|---|
| 兜底止损 | 任何时候 | 跌破 SL2（最近向下笔低点）立即止损 |
| 浮盈 >= 10% | 从最高点回撤 15% | 启动跟踪止损，锁定部分利润 |
| 浮盈 >= 25% | 从最高点回撤 10% | 收紧跟踪止损，保护大部分利润 |
| 最大持有 | 60 天 | 到期按收盘价退出 |
