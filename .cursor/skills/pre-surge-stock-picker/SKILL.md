# 主升前 CZSC 特征每日选股

## 工作流

执行以下三步，每步完成后再进入下一步：

### Step 1: 同步最新数据

```bash
PYTHONUNBUFFERED=1 /home/lovelyzzc/czsc/.venv/bin/python /home/lovelyzzc/czsc/scripts/_sync_daily_data.py
```

确认输出包含"完成"或"数据已是最新"后继续。

### Step 2: 运行每日扫描

```bash
PYTHONUNBUFFERED=1 /home/lovelyzzc/czsc/.venv/bin/python /home/lovelyzzc/czsc/.cursor/skills/pre-surge-stock-picker/scripts/daily_scan.py
```

扫描全 A 股约 10 秒，输出加权得分 >= 5.0 的标的。

### Step 3: 汇报结果

将扫描结果整理为表格呈现给用户，包含以下列：

| 列 | 说明 |
|---|---|
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 行业 | 所属行业 |
| 收盘价 | 最新收盘价 |
| 得分 | 加权总分（满分 6.5） |
| C1~C7 | 各项原始得分（0/1） |
| MA状态 | 是否多头排列 |
| **推荐止损** | **SL2 笔结构止损价（最近向下笔低点）** |
| 止损幅度% | (收盘价 - 推荐止损) / 收盘价，直观展示风险 |
| 止损_中枢ZD | 前一中枢下沿（辅助参考） |
| 止损_ATR | 收盘价 - 2x ATR20（辅助参考） |

得分 >= 5.5 的重点标注，5.0 的作为观察池。止损幅度为负值的标的（当前价已跌破笔低点）不建议介入。

## 加权打分说明

权重通过 149,399 条历史样本（单因子分析 + 逻辑回归）优化得出：

| 编号 | 条件 | 含义 | 权重 | 预测能力 |
|---|---|---|---|---|
| C1 | 中枢上移 | 最近 3 个中枢 ZD 递增 | **1.0** | 正向 +0.43% |
| C2 | 窄幅收敛 | 最近中枢宽度 < 10% | 0.5 | 中性 |
| C3 | 笔力突增 | 向上笔力度 > 前均值 x 1.5 | 0.5 | 弱 |
| C4 | 下笔递减 | 向下笔力度递减且幅度 < 12% | 0.5 | 弱 |
| C5 | DIF零轴 | DIF 绝对值 < 0.5 | 0.5 | 中性 |
| C6 | 缩放量 | 前期量比 < 0.8，当前 > 1.2 | **3.0** | **最强 +1.47%** |
| C7 | 不回中枢 | 下跌笔低点 > 前一中枢 ZG | 0.5 | 弱 |

加权满分 6.5，入选阈值 5.0。实际上要求 C6（缩放量）必须满足，再加 C1 或多项其他特征。

## 止损策略与回测表现

**推荐止损：SL2 笔结构止损（最近向下笔低点）**

加权打分 + SL2 止损的回测表现（7,849 笔交易）：

| 指标 | 等权基准 | **加权优化** |
|---|---|---|
| 年化收益 | 82.9% | **133.9%** |
| 夏普比率 | 3.02 | **2.61** |
| 最大回撤 | 12.1% | **18.4%** |
| 卡玛比率 | 6.85 | **7.28** |
| 盈亏比 | 2.42 | **2.19** |
| 平均收益 | 2.27% | **2.65%** |

止损逻辑：价格跌破最近向下笔的低点时退出，意味着上升笔结构被打破。

辅助参考（仍在输出中提供）：
- **中枢止损**：前一个中枢下沿 ZD，跌破意味着中枢上移结构破坏
- **ATR止损**：收盘价 - 2 倍 ATR20，适合短线风控
